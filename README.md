# docker-xmrig

Docker implementation of MoneroOcean xmrig fork

https://github.com/MoneroOcean/xmrig


## Usage
Pull:

    docker pull mcjon3z/xmrig
    
Run:

    docker run -it --rm mcjon3z/xmrig xmrig <OPTIONS>
    
or
    
    ./xmrig.sh <OPTIONS>
